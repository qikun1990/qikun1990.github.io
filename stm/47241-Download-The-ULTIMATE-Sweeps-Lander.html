    Download: The ULTIMATE Sweeps Lander

    Well, maybe not really the ultimate sweeps lander but at least a very good one for the lazy ones among us

    I guess most of you know the gift box lander.

    Probably also most of you know the version where you can edit the different values all comfortable in the JS file so that customizing the lander for different products is much easier.

    I give you my version of the lander now where it´s even easier than the JS version.

    In my version you can edit the values with URL parameters so you don´t even need to make changes in the files anymore when you want to run different products or call out different brands.

    Let me show you what I mean.

    When you open the lander just the way it is it looks like this







    As you can see I use the {device_brand} token from my tracker to callout the "sponsor".
    That way the users device brand will be shown on the modal and the lander itself.

    I use Binom so the token can be different depending on the tracker you use.

    By default the values for the brand and product that you want to promote are set to "Brand" and "Product".
    I know, not very creative but what of it

    Now let´s say you want to promote a Samsung Galaxy S20.

    Then you just add ?bra=Samsung&pro=Galaxy+S20 to the URL and see what happens





    You can also change the "sponsor" when you add &spo=SPONSOR to the URL

    When you want to add a logo to the top left corner or a header to the top right corner just put the files in the folder with the landing page and write the names in the javascript.js file



    To add a logo change line 26

    Code:
    var a = "YOU_LOGO.png"; // logo relative path
    To add a image as right header change line 27

    Code:
    var a2 = "YOUR_HEADER.png"; // header right icon relative path
    And here is the link to the DOWNLOAD

    Have fun.

    P.s.: Don´t forget to change the click URL in the index.html
