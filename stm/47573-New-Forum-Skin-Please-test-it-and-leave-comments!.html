    New Forum Skin - Please test it and leave comments!

    NEW FORUM SKIN AVAILABLE


    Most of you probably don't even know that there is an option to switch the skin/theme of the forum, am I right?

    For now, the waste majority is using the default theme that looks like this:





    We felt like it was about time for a small makeover, so thanks to @shishev, we now have a new forum skin available

    It uses a more current format and design elements, but still using the color scheme we're all used to.

    Just like is the case with anything new, there MIGHT be some small tweaks required, so we would like to ask you guys to help us test-drive this new theme and leave your feedback here in this thread.

    The new skin feels quite different from what you're likely used to, so give it some time to get used to it.

    I'm using it for the past two weeks or so and have to say I like it more than the old one


    How to switch?


    There is a "selector" at the bottom left. The current default theme is named "STM V2 - Large Fonts" and the new one is "STM V3".

    Here is a screen to show you exactly where to find the selector... the red arrow points to the new skin






    And before you switch, here is a quick look at the new skin:






    Please, switch to the new skin and try it out and let us know what you think ... even if you don't like it and you decide to go back to the old one, that's fine.

    Just tell us why you did that

    Thanks,
