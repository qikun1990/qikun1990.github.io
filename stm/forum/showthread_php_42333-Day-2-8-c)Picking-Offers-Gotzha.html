    Day 2-8: c)Picking Offers - Gotzha

    ***********************************

    ACTION

    This lesson will be very brief - Gotzha's interface is similar to Clickdealer's because they both run on the same affiliate network software - CAKE.

    Like Clickdealer, Gotzha has a good variety of offers in different offer verticals. Sweepstakes is one of their strong verticals - which is one of the main reasons I'm wanting you to sign up to them. When we get around to running offers using landing pages, sweepstakes would be the vertical I'd recommend for you to start with - it's a great vertical for beginners.

    If you haven't gotten approved by Gotzha yet, feel free to skip to the next lesson. As was mentioned before, all we really need is for you to be approved for ONE offer at ONE network.

    Without further ado - let's pick out an offer on Gotzha!






    -Log into your Gotzha account.

    -In the left menu click on "Offers".

    -Specify the following search criteria: Set "Payout" "Max" to "2" (uncheck "Use My Currency" - we want offers with payouts of < $2 USD), "All Verticals" = "Mobile Content", "All Tags" = "1-click flow", "All Statuses" to "Public" (this means EVERYONE has permission to run them without having to apply and wait for approval).



    -Choose an offer. Avoid adult/sexy/erotic ones. Pick something for a developing country. Games and videos (non-adult) has a chance to convert when direct-linked, so would make good choices. (We've been through all this before!) Click on the offer name of your chosen offer.

    -In the pop up that results, choose a media type - they don't have pop as an option but choosing "Contextual" would be fine (again, hardly any networks ever checks these) and click "Submit".



    -Since the offer is "Public" you'll be automatically approved to run it. A window containing the offer details would automatically pop up. The 2 tabs you really need to browse are "Basic Info" - which sometimes contain valuable information such as payout and where and how the offer can and cannot be run - and the "Creatives" tab which contains your offer link(s).



    (Note: If you somehow need to locate your offer again, you'll need to change the "All Statuses" filter to "Active" in order to see it because it's now been approved. Either that, or set it back to "All Statuses" and paste the offer ID into the "Search Here For Offers..." field.)

    -As we've done for the previous networks (you can browse the Mobidea lesson to go over the part where you can download a spreadsheet to keep track of offers), record your offer's details in the offers spreadsheet. Like so:



    This particular offer does not specify which OSs are accepted. I would assume that ALL OSs are accepted, but when questions like these arise, it's best to check with the AM.



    ***********************************

    EXPLANATION

    No further explanation necessary.



    ***********************************

    READING

    No additional reading required.



    ***********************************

    Now that we've been approved for at least one offer, we're ready to set up some campaigns in your chosen tracker!
